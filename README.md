# 🌿 Green Soft - IoT 통합 관제 및 비즈니스 플랫폼
> **산업 현장 모니터링과 이커머스 기능을 결합한 통합 비즈니스 플랫폼**

Green Soft는 실시간 데이터 모니터링이라는 기술적 가치와 제품 거래를 위한 커머스 기능을 유기적으로 연결한 프로젝트입니다. 효율적인 운영을 위한 관리자 대시보드와 사용자 편의를 위한 외부 API 연동에 집중하여 개발했습니다.

## ✨ Key Features

### 1. 쇼핑몰 시스템 및 실시간 결제
* **PortOne API** 연동을 통한 안전하고 빠른 실시간 결제 및 검증 시스템 구축
* 상품 등록, 실시간 재고 관리, 상품 리뷰 등 커머스 핵심 기능 구현

### 2. 위치 기반 시각화 및 공공데이터 연동
* **소방청 화재 통계 API**를 활용한 연도별/지역별 데이터 조회 및 시각화
* **Naver Maps API**를 통한 업체 위치 정보 제공 및 목적지 자동 입력 길찾기 기능 구현

### 3. 지능형 커뮤니티 및 관리 기능
* **자동 블라인드 시스템:** 신고 3회 누적 시 게시글/댓글 자동 비노출 처리 및 관리자 승인 프로세스 구축
* 포토갤러리(무한 스크롤), 공지사항, 자료실 등 통합 커뮤니티 모듈 개발

### 4. 실시간 운영 관리 대시보드
* **주요 운영 지표 시각화:** 배송 준비, 신규 문의, 신고 현황 등을 실시간 수치로 요약 출력
* 운영 업무 우선순위 파악을 돕는 직관적인 UI 설계로 관리 효율화 달성

## 📈 Learning & Achievement
* **최신 스택 활용 및 협업:** **Java 21**과 **Spring Boot** 환경에서 팀원들과의 코드 통합 및 형상 관리를 경험하며 실무적인 협업 능력을 길렀습니다.
* **문제 해결 및 UX 최적화:** 무한 스크롤, 목적지 자동 입력 등 사용자의 불편함을 줄이기 위한 기술적 고민을 통해 서비스 완성도를 높였습니다.
* **운영 관점의 개발:** 대시보드 구축을 통해 단순 기능 구현을 넘어, 시스템 운영 효율성까지 고려하는 개발자로서의 시야를 확보했습니다.
